# Agentic-News-Analysis-System


## Overview

This project implements an NLP-powered Agentic AI system capable of autonomous planning, memory management, and dynamic tool selection.

The system integrates:

* News Classification
* Named Entity Recognition (NER)
* Planning Component
* Memory Module
* Tool Selection Mechanism

## Features

* Perception Layer
* Rule-Based Planner
* Dynamic Tool Selection
* Named Entity Recognition using BERT
* News Classification
* Memory Storage
* End-to-End Agent Workflow

## Architecture

The system follows:

User Query → Planner → Tool Selection → NLP Tools → Memory → Final Response

## NLP Tasks

### 1. News Classification

Classifies news into:

* Business
* Sports
* World
* Science/Technology

### 2. Named Entity Recognition

Extracts:

* Persons
* Organizations
* Locations
